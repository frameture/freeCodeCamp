Timestamp Microservice

Usage:

Timestamp microservice takes an argument through its URL and outputs a JSON object with two properties: unix and natural

The argument might be an unix number which will be converted into natural representation of a date or a string with normal date representation that will be converted to unix number.

Examples of URL :
https://frameture.github.io/timestamp-microservice-front-end/1450137600 https://frameture.github.io/timestamp-microservice-front-end/December%2015,%202015
Output :
{{ '{' }} "unix": 1450137600, "natural": "December 15 2015" {{ '}' }}
